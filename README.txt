The Peachtree extras plugin.

== Description ==

This is the "Peachtree Analytics" WordPress plugin, which helps your WordPress installation give the PeachTree Analytics platform more relevant information about how users interact with your site.

== Installation ==

e.g.

1. Upload the plugin files to the your WordPress plugins page
1. Activate the plugin
1. Add your Peachtree site ID to the new setting in Settings->General
